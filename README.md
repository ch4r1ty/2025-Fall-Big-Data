Welcome to our project: NYC Crime Data Analysis 2024 

In this project, we use big data analytics and machine learning to analyze and predict crime patterns in New York City.
We mainly use Python, MongoDB, Pandas, Matplotlib, Seaborn, Plotly, and SARIMA for data processing, visualization, and forecasting.

Steps to reproduce results:
1. Download dataset

Dataset: NYPD Complaint Data Historic (2024)

Source: NYC Open Data

URL: https://data.cityofnewyork.us/Public-Safety/NYPD-Complaint-Data-Historic/qgea-i56i

2. Set up environment

Install Python (3.9+ recommended)

Install MongoDB (local)

Install required libraries:

pip install pandas numpy pymongo matplotlib seaborn plotly statsmodels

3. Run data pipeline

Run the scripts in the following order:

python download_2024_robust.py
python clean_data.py
python check_data_quality.py

4. Run analysis notebook

Open the Jupyter Notebook file:

jupyter notebook
